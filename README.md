## IDX AI Stock Screener (Blueprint & Implementation)

Sistem ini adalah implementasi Stock Screener otomatis untuk pasar saham Indonesia (IDX). Menggunakan Python untuk kalkulasi berat (Heavy Lifting) dan Next.js untuk visualisasi.

## 1. Struktur Folder

/idx-screener
├── screener_engine.py   # Backend Logic (Python)
├── public_screener_data.json # Output Data (Generated by Python)
└── dashboard/           # Next.js Project
    └── src/app/page.tsx # Frontend UI


## 2. Cara Menjalankan (Python Backend)

Bagian ini bertugas mengambil data dari API, melakukan simulasi kalkulasi teknikal (MA, RSI, MACD), dan memberi skor.

Pastikan Python terinstal.

Install library yang dibutuhkan:

pip install requests pandas numpy


Jalankan script:

python screener_engine.py


Script akan menghasilkan file public_screener_data.json. File ini berisi daftar saham yang sudah di-scoring (0-100).

## 3. Cara Menjalankan (Next.js Frontend)

Bagian ini memvisualisasikan data JSON menjadi dashboard interaktif.

Buat project Next.js baru (jika belum ada):

npx create-next-app@latest dashboard
cd dashboard


Copy kode dari file page.tsx ke dalam src/app/page.tsx.

Pastikan install lucide-react untuk icon:

npm install lucide-react


Jalankan server dev:

npm run dev


Buka http://localhost:3000.

## 4. Catatan Penting Mengenai Data

Limitasi API Publik:
API pasardana.id endpoint GetAll hanya memberikan data snapshot (Harga Terakhir, Perubahan Hari Ini).

Solusi di Script Ini:
Agar script screener_engine.py bisa didemokan tanpa API Key berbayar (yang diperlukan untuk menarik history candle OHLCV 200 hari ke belakang), script ini menggunakan fungsi generate_synthetic_history.

Untuk Production:
Anda harus mengganti fungsi generate_synthetic_history di Python dengan panggilan API sungguhan ke penyedia data candle history (seperti GoAPI, Yahoo Finance, atau API sekuritas Anda) untuk mendapatkan akurasi RSI dan MACD yang presisi.
